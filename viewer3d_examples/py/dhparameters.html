DH parameters example (Python version)

This example shows how to define a robot using modified DH parameters.

The user can click on the sphere at the end of the robot to activate some controls to manipulate it using Inverse Kinematics (ie. change its position and orientation).

Likewise, clicking and dragging on any link of the robot allows to use IK to manipulate it.

It is also possible to change the position of the joints by hovering on the links of the robot and using the mouse wheel.

from viewer3d import Viewer3D, RobotBuilder from js import document import math document.getElementById('loader-container').style.display = 'none'; # Create the Viewer3D viewer3D = Viewer3D(document.getElementById('viewer3d')) # Define the robot using DH parameters builder = RobotBuilder.new('robot'); builder.q = [0, 0, 0, 0, 0, 0, 0, 0] # Angle about previous z builder.q_offset = [0, 0, 0, 0, 0, 0, 0, 0, 0] # Offset on articulatory joints builder.alpha = [0, -math.pi/2, math.pi/2, math.pi/2, -math.pi/2, math.pi/2, math.pi/2, 0] # Angle about common normal builder.d = [0.333, 0, 0.316, 0, 0.384, 0, 0, 0.107] # Offset along previous z to the common normal builder.r = [0, 0, 0, 0.0825, -0.0825, 0, 0.088, 0] # Length of the common normal builder.defaultPose = [0, 0, 0, -math.pi/2, 0, math.pi/2, 0] builder.colors = [ # As many as you want, links will alternate between those colors '0.9 0.9 0.9 1', '0.7 0.7 0.7 1', ] # Load the scene and retrieve the robot viewer3D.loadScene('/scenes/empty.xml', [builder]) robot = viewer3D.createRobot('robot', builder.getConfiguration()) # Start the physics simulation viewer3D.physicsSimulatorPaused = False
