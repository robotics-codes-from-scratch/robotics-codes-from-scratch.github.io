RCFS aims to study and test learning and optimization techniques in robotics through simple 2D and 3D examples (see About page).

The menu on the top-left corner provides sandbox examples, interactive exercises, and reading material.

RCFS is composed of three parts:

  • Interactive sandbox examples and exercises that can be run directly on this website (some GIFs are shown below). All examples are interactive, either by using the code snippets that can be modified, or by changing parameters with the mouse (changing the location of the objects to be picked by the robot, perturbing the robot with the mouse pointer, adjusting the control parameters with a set of sliders, etc.)
  • Source codes provided in a variety of programming languages , including Python, Matlab (with full compatibility with GNU Octave), C++ and Julia.
  • An accompanying PDF containing the corresponding descriptions of the algorithms.
